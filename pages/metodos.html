Métodos Importantes


toString()

Esse método converte um array em uma string com os valores do array separados por vírgula.


join()

Esse método é semelhante ao toString(), mas neste é possível especificar o separador.


pop()

Permite que você remova o último elemento e retorna o valor do elemento que foi retirado.


push()

Permite que você adicione um novo elemento ao final do array e retorna o novo tamanho do array.


shift()

Esse método é semelhante ao pop(), mas neste é ocorre a remoção do primeiro elemento e retorna o valor do elemento retirado.


unshift()

Esse método é semelhante ao push(), mas neste é ocorre a adição do primeiro elemento e retorna o novo tamanho do array.


length

Esse método retorna o tamanho do array.


delete()

Esse método deleta elementos do array, mas deixa espaços "undefined" de onde o elemento foi retirado.


concat()

Esse método concatena arrays, criando um novo array com os elementos dos arrays que foram concatenados.


splice()

Esse método pode ser usado para adicionar e remover elementos de um array e retorna um array com os elementos deletados.
- O primeiro parâmetro corresponde à posição onde os novos elementos devem ser adicionados;
- O segundo parâmetro define quantos elementos serão removidos a partir da posição;
- O resto dos parâmetros define os novos elementos que serão adicionados.


slice()

Esse método "fatia" arrays, criando um novo array com os elementos dos arrays a partir do índice indicado.
Esse método não deleta elementos do array original, apenas gera um novo com elementos selecionados dos original


sort()

Esse método ordena um array por ordem alfabética.
