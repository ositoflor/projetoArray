Array


O que é e por que usar um array?

O array é uma variável especial que suporta mais de um valor. O array pode armazenar muitos valores em apenas uma variável onde cada valor possui seu index e podemos acessá-los refereciando o número do index.
Arrays são um tipo especial de objetos com indexes numerados.

Como reconhecer um array?

Podemos usar o método Array.isArray(nomeArray) ou instanceof. Ambos retornam true se for um array.
